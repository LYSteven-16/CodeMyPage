## CodeMyPage 是什么？

CodeMyPage 是一款**拖拽式网页编辑器**，专为教育工作者和内容创作者设计。无需任何编程知识，即可构建互动教学页面。

### 你可以创建的内容

- 📝 **文本与媒体** - 标题、段落、图片、视频、音频
- 📋 **互动元素** - 测验、填空、选择题、判断题
- 🔢 **排序练习** - 拖拽排序练习
- 🎨 **创作工具** - 绘图板，支持自由绘画
- 📊 **学习辅助** - 进度条、时间线、清单
- ✅ **反馈组件** - 答题卡、分数展示、答案解析

## ⚠️ 导出前必读

### 导出做了什么

当你点击**导出**按钮时，编辑器会创建一个可以在任何电脑上运行的文件。这很方便，但也需要你小心使用。

**下面来解释一下发生了什么：**

- **导出的文件是一个小程序** - 它包含了你的所有内容，以及显示这些内容所需的工具
- **它需要联网下载代码** - 文件打开时会自动从网上下载 React（一个代码库）
- **你的内容就藏在文件里** - 你创建的所有内容都会被存储在这个文件中

**如何安全使用：**

- 🔒 只从**你信任的网站**使用这个编辑器（比如官方 GitHub Pages）
- 🔒 **不要打开陌生人发来的导出文件** - 它们可能包含恶意代码
- 🔒 如果收到别人发来的文件，先用杀毒软件**扫描一下**
- 🔒 只从 **https://** 开头的网站下载编辑器（不要用 http://）

**危险信号：**

- 有人突然给你发送了一个导出文件
- 文件来自一个你从未听说过的网站
- 你的杀毒软件发出警告

### 导出技术说明

#### 导出流程

当触发交互式导出功能时，编辑器执行以下操作：

1. **构建阶段**：服务器端的 Vite 构建系统将包含所有项目组件、样式和配置的完整 React 应用编译为优化的 JavaScript 包
2. **打包阶段**：编译后的 JavaScript 和 CSS 资源直接嵌入到自包含的 HTML 文档中
3. **运行阶段**：导出的 HTML 文件从 CDN（unpkg.com）加载 React 和 ReactDOM 库，并在客户端初始化应用程序

#### 潜在安全风险

| 风险类别 | 描述 | 影响程度 |
|---------|------|---------|
| **代码注入** | 如果编辑器的源代码被恶意修改，导出的文件可能包含嵌入的恶意代码 | 高 - 可能获得完整系统访问权限 |
| **供应链攻击** | 导出的文件依赖外部 CDN 获取 React/ReactDOM；CDN 被攻陷可能导致恶意代码注入 | 高 - 影响所有导出的文件 |
| **数据泄露** | 导出的文件包含完整的项目数据，包括所有组件和配置 | 中 - 知识产权披露 |
| **沙箱逃逸** | 虽然导出的文件在浏览器沙箱中运行，但结合其他漏洞可能利用浏览器缺陷 | 中 - 取决于浏览器安全性 |

#### 安全建议

- 通过 git 提交签名或校验和验证源代码完整性
- 在生产部署中对 CDN 依赖使用子资源完整性（SRI）哈希
- 在托管导出文件时实施内容安全策略（CSP）头
- 对于敏感部署，考虑自托管 React 依赖
- 分发前使用静态分析工具审计导出的文件

#### 技术规格

- **构建系统**：Vite 配合自定义导出配置
- **输出格式**：单个自包含 HTML 文件
- **运行时依赖**：React 18.x, ReactDOM 18.x（从 CDN 加载）
- **浏览器兼容性**：Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
